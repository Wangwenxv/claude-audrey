# Product

## Register

product

## Users

Audrey Hall is used by people who want a desktop companion that can also act as a practical Claude-powered work agent. They are usually in a focused desktop workflow: chatting naturally, asking for help, granting permissions, watching tool progress, and expecting the assistant to feel present without blocking real work.

## Product Purpose

The product gives Audrey Hall a usable chat surface and agent control center inside a desktop pet application. Success means the conversation feels characterful and intimate when chatting, while agent activity feels clear, trustworthy, and operational when work is underway.

## Brand Personality

Elegant, mysterious, capable. Audrey should feel like a role-play companion with a distinct presence, not a generic bot, while the working interface should carry the confidence of a professional desktop agent app.

## Anti-references

Avoid generic web-chat layouts, toy-like status spam, decorative dashboards, and interfaces that hide real tool progress behind vague loading copy. Avoid making the agent feel either purely fictional or purely utilitarian; both modes must coexist.

## Design Principles

Conversation should feel close and social, with clear sender identity and readable message rhythm.
Working state should feel like a desktop agent console: concrete, inspectable, and calm.
Role-play tone should enhance trust, not obscure permissions, errors, or progress.
Preserve the existing Audrey Hall visual language before adding new motifs.
Every status surface should answer what is happening, whether it needs user attention, and what changed.

## Accessibility & Inclusion

Default to readable contrast, familiar desktop controls, keyboard-friendly chat behavior, and motion that communicates state rather than decoration. Keep reduced-motion users in mind when adding animation or pulsing indicators.
